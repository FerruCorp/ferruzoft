Este ejemplo va a tratar sobre el hoisting.
            
                Comenzamos por un ejemplo:
                    console.log(a);
                Si mandamos eso en consola nos mostrará Uncaught ReferenceError: a is not defined
                pero si hacemos lo siguiente:
                    console.log(a);
                    var a;
            
                Esto va a mostrar que la variable se encuentra indefinida o en inglés undefined, y no sé mostrará
                el mensaje: "Uncaught ReferenceError:...". Esto es debido a que javascript detecta la variable o función
                declarada sin importar el orden en el código pero aun así la toma como undefined dado que no tiene una 
                valor u objeto asignado.                
                
                Aun asi hagamos lo siguiente:
                    console.log(a);
                    var a = 3;
                De todas formas indicará que la variable "a" se encuentra undefined.    
            
                Ahora vamos a probar con funciones para conocer su manejo.
            
                Comenzamos por un ejemplo:
                    console.log(funcionUno);
                    console.log(funcionDos);
                    function funcionUno(){
                        //mi código
                    }                    
                    var funcionDos = function(){
                        //mi código
                    }
                Si mandamos eso en consola nos mostrará:
                    function funcionUno(){
                        //mi código
                    }    
                    undefined
                Ahora si hacemos lo siguiente:
                    var fac = factorial(5);
                    
                    var factorial = function(param){
                        console.log('factorial!');
                    }
            
                Esto va a mostrar un error tipo: Uncaught TypeError in dicando que la propiedad factorial
                no es una función.
