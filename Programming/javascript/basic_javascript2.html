Este ejemplo va a tratar sobre la diferencia entre scope y clousure
                en javascript.
            
                 Vamos a comenzar por Scope el cual se divide en 2 partes: Global  y Local.
                 El Scope global trata de una variable que se puede acceder tanto fuera como dentro
                 de una función. Un código de ejemplo es el siguiente:
                    var a = 1;
                    function global() {
                        console.log('Accedo desde la función global: ' +a);
                    }
                    global();
                    console.log('Accedo desde cualquier otra parte: ' + a);
            
                Ahora veremos el ejemplo de un Scope Local. Éste solo puede ser accedido desde detro de una misma función y/o
                bloque de código. Por ejemplo:
                    function local() {
                        var a = 2;
                        console.log('Accedo desde la función local:' + a);
                    }
                    local();
                    console.log('Intento acceder a la variable declarada dentro de la función local pero nos dice que la variable \"b"\ es: ' + Uncaught ReferenceError: b is not defined ); 
            
                Esto es debido que la variable b se encuentra declarada dentro de la función local y no se puede acceder a ella
                fuera de la función.   
            
                Un closure es una función que es libre de variables, esto quiere decir que las variables de la función padre funcionan, pero el closure no tiene variables propias. Ejemplo:
                    
                    function padre() {
                        var a = 1;
                        funcion closure() {
                            console.log(a);
                        }
                        closure();
                    }
                    padre();
                    
                La función interna es un clousure y solo esta diponible en la función padre.    
            
 
                Ahora un ejemplo un poco mas "complejo"
                    function Suma(a) {
                        return funcion(b) {
                                return a + b
                        }
                    }
                    var sumar5 = Suma(5);
                    var sumar10 = Suma(10);
                    console.log(sumar5(15));
                    console.log(sumar10(15));
