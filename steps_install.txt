1) gem install watir-webdiver
2) gem install testgen
3) gem install bundler
4) crear carpeta
5) dar directorio consola
6) luego ingresar:
testgen project [project_name] --pageobject-driver:watir
7) Despues ingresar al [project_name] por consola
8) Finalmente colocar bundle install en consola

* correr por consola (estar en la carpeta del proyecto):
bundle exec cucumber -p default [file]/[file_name]
